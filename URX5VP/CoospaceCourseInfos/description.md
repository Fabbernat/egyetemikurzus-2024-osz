# Description (public)

A kurzus célja, hogy a hallgató megismerkedjen a modern C# nyelvi elemeivel és lehetőségeivel, úgy, ahogy azt a nagyobb
projekteke fejlesztése során is alkalmazzák. A kurzus elsősorban a C# nyelv lehetőségeire és ajánlott
alkalmazásfejlesztési alapismeretekre, elvekre fókuszál konkrét grafikus keretrendszer vagy adatbáziskezelő rendszer
ismertetése nélkül.
Feltételezett előismeretek, javasolt előismereti tárgyak (opcionális):
Programozás alapjai (C programozás), Programozás I (Java programozás)
